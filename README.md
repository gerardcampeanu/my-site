# Victor Family Sites

A very small static website for Victor, Gerard, and Tomas.

The site can be opened locally by opening `index.html` in a browser. It also
supports simple subdomain-style routing:

- `victor.your-domain.com` shows Victor's page
- `gerard.your-domain.com` shows Gerard's page
- `tomas.your-domain.com` shows Tomas's page

For local testing, use the links on the page or add `?site=victor`,
`?site=gerard`, or `?site=tomas` to the URL.
